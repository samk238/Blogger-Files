We can install WLS 12.1.3.0.0 in two modes :

- GUI mode

- Silent mode

 

Note:

- We donot have the console mode option anymore

- JDK is not bundled along with the installer, so install JDK first and then install Weblogic Server.

- If first time

    - Oracle Inventory is created first and then WLS is installed. If Oracle Inventory is already present then the new installation details would be appended to this file.

    - Oracle Inventory is similar to the registry.xml file that used to get created in older WLS versions. Oracle Central Inventory keeps track of all the Oracle Software products installed on all the Oracle homes on your system, provided the products were installed using Oralce Universal Installer.

 

STEPS:

**- Install JDK 1.7

**- Download the WLS installer fmw_12.1.3.0.0_wls.jar

**- Run the following command :  java -jar fmw_12.1.3.0.0_wls.jar

 

**-Oracle_Home

 

**-

1. Weblogic Server Installation - For WebLogic Server and Coherence standard installation topology - no server examples

2. Coherence Installation - to deploy and manage Coherence applications using the WebLogic Management Framework. - no server examples

3. Complete Installation - WLS + Coherence + server examples

 

**Security updates

 

Creating Domain:

  <ORACLE_HOME>/wlserver/common/bin - ./config.sh

 

    Select Create Domain Using Product Templates.

    All Templates:

      Under Available Templates, Basic WebLogic Server Domain is selected and cannot be deselected. It is the only required template.

      Click Next

 

-> Administrative account - uid, password

               Domain Mode, select Production.

               JDK, select Oracle HotSpot

 

Administration Server, Node Manager, and Managed Servers, Clusters and Coherence

 

AdminnServer: host name or IP address of the machine for Listen Addres, Port,          SSL Option deselected.

 

Node Manager   (Leave the default value of Per Domain Default Location)

    Credentials

    Note: The credentials for Node Manager do not have to match the administrator account credentials.

 

Managed Servers page, click Add to display

  -> Enter Server Name.

  -> Enter the host name or IP address of the machine for the Listen Address.

  -> Listen Port is 7003

  -> Leave Enable SSL deselected.

 

Clusters page, click Next.

 

Machines  - ip address and 5556 port
